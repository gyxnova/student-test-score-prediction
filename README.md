# Student Test Score Prediction

This project predicts student exam scores using tabular data and machine learning models.

## Approach
- Data cleaning and EDA
- Feature engineering
- Model training with LightGBM and CatBoost
- Cross-validation
- Ensemble (blending)

## Result
Best CV RMSE achieved: ~8.73

## Tools
Python, pandas, numpy, scikit-learn, LightGBM, CatBoost
